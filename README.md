# Password Strength Checker

A simple Python tool that evaluates the strength of a password based on common security rules.

## Features

- Checks password length
- Detects uppercase and lowercase characters
- Detects numbers
- Detects special characters
- Classifies password strength (Weak, Moderate, Strong)

## Technologies

Python
Regular Expressions (re module)

## Security Relevance

Weak passwords are a common cause of security breaches. This tool demonstrates how password policies can help prevent brute force attacks and credential compromise.
